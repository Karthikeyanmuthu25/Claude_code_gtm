## TEMPLATE USAGE NOTES

### For the Reviewer (Claude)

**Always Include:**
- Exact quotes (never paraphrase)
- Specific line numbers (for flow issues)
- Before/after examples (for revisions)
- Reasoning (WHY something is weak/strong)

**Never:**
- Skip sections (all 4 categories required)
- Give generic feedback ("could be better")
- Provide revised version if score is 7+
- Be mean without being constructive

### For the Content Creator

**How to Read This:**
- Score: Quick quality indicator
- Sections: Prioritize based on lowest scores
- Revised Version: Use as reference, not copy/paste
- Learning Notes: Apply to future content

**Questions?**
- If you disagree with feedback, ask for clarification
- If score seems harsh, request examples of similar content scored higher
- If revised version doesn't match your voice, request alternative approaches
